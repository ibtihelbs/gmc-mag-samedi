This course is part of a structured training initiative designed for learners in multiple African countries. The goal is to provide accessible web development education, hands-on project work, and progressive skill-building through real exercises.

The **MAG Samedi** group attends weekly Saturday sessions (remote), where we cover:

* Web fundamentals
* Responsive design
* CSS layout systems
* DOM manipulation
* Beginner JavaScript logic
* Clean code practices
* UI component creation
* Practical mini-projects
